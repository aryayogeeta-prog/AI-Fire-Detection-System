AI-Based Fire Detection System
## Overview

This project uses Artificial Intelligence and IoT to detect fire and smoke in real time. The system combines a camera, MQ-2 gas sensor, DHT11 temperature sensor, and NodeMCU ESP8266 to provide early fire detection and alert generation.

## Features
Real-time fire and smoke detection
AI-based image analysis using YOLO
Gas and temperature monitoring
Alarm and alert system
Snapshot capture
Streamlit dashboard
